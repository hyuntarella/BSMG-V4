# 3모드 수동 테스트 체크리스트

## 사무실 모드
- [ ] 실제 타이핑 시 하이라이트 반응 속도 (체감 지연 없는지)
- [ ] 한글 입력 중 IME 조합 상태에서 하이라이트 깨지지 않는지
- [ ] "바탕 500" 입력 → 미리보기 하이라이트
- [ ] Enter → 시트 반영 + 대화 로그 기록
- [ ] ESC → 미리보기 취소
- [ ] "바탕 500 1000 200" → 재/노/경 3개 필드 반영
- [ ] "바탕 500 1000" → 재/노 2개 필드 반영
- [ ] 위 화살표 → 이전 명령 복원
- [ ] 멀티라인 붙여넣기 → 줄 단위 처리
- [ ] "전체 재 +10%" → 일괄 조정
- [ ] 대화 로그에 ⌨ 아이콘 표시

## 현장 모드
- [ ] 마이크 → 실시간 텍스트 표시
- [ ] 말하는 중 행/셀 하이라이트 동작
- [ ] "넣어" → 즉시 반영
- [ ] "바꿔줘" / "올려" 등 종결어미 → 즉시 반영
- [ ] "아니" → 0.2초 롤백
- [ ] "아니 600" → 롤백 + 재적용
- [ ] "바탕 500 1000 200" 음성 → 재/노/경 반영
- [ ] 필드 자동 추론 동작
- [ ] "도" 패턴 계승 동작
- [ ] 불완전 명령 버퍼 축적 동작
- [ ] 효과음 재생 (딩/둥/되감기)
- [ ] 이상치 입력 시 로그 경고

## 운전 모드
- [ ] 모든 종결어미로 실행 ("넣어" 포함)
- [ ] 무음 2초 → 자동 실행
- [ ] TTS: 모든 실행 결과 읽기
- [ ] TTS: "현황" → 상태 요약
- [ ] TTS: "총액" → 총액 읽기
- [ ] TTS: 이상치 경고
- [ ] TTS: "아니" → 롤백 확인 읽기
- [ ] TTS: 확신도 낮을 때 되물음
- [ ] TTS 재생 중 마이크 입력 안 받는지

## 교정 시스템 (모든 모드)
- [ ] 👎 → 인라인 교정 입력
- [ ] 교정 입력 → DB 저장 확인 (Supabase)
- [ ] 교정 이력이 다음 명령에 반영되는지
