현재 상태:
- handleVoiceCommands에 'read_summary'와 'read_margin' 시스템 명령이 구현되어 있음.
- 하지만 이 경로는 LLM을 거쳐야 함: STT → LLM(modify 프롬프트) → commands[{action: 'read_summary'}] → handleVoiceCommands.
- 문제: LLM이 "현재 상태 알려줘"를 시스템 명령으로 파싱하지 않고 modify 명령으로 해석할 수 있음.
- 또한 LLM 호출 자체가 1-3초 지연을 발생시킴.

해결 방향:
- onSttText에서 "현재 상태", "상태 알려", "마진 얼마" 키워드를 직접 감지.
- LLM 호출 없이 estimate 데이터에서 직접 요약 텍스트 생성.
- voicePlayTtsRef.current()로 TTS 출력.
- return true로 LLM 건너뜀.

수정 범위: hooks/useEstimateVoice.ts의 onSttText 콜백만 수정.

   ```typescript
   // "현재 상태" / "상태 알려" / "요약" 키워드
   if (/현재\s*상태|상태\s*알려|요약/.test(normalized)) {
     const sheetIdx = activeSheetIndex >= 0 ? activeSheetIndex : 0
     const sheet = estimate.sheets[sheetIdx]
     if (sheet) {
       const total = sheet.grand_total ?? 0
       const itemCount = sheet.items.length
       const margin = callbacksRef.current.getSheetMargin(sheetIdx)
       const msg = `${sheet.type} 면적 ${estimate.m2}제곱미터, 공종 ${itemCount}개, 총액 ${Math.round(total / 10000).toLocaleString()}만원, 마진 ${Math.round(margin)}퍼센트.`
       callbacksRef.current.addLog('assistant', msg)
       voicePlayTtsRef.current(msg)
     }
     return true  // LLM 건너뜀
   }

   // "마진 얼마" / "마진" 단독 키워드
   if (/마진\s*얼마|^마진$/.test(normalized)) {
     const sheetIdx = activeSheetIndex >= 0 ? activeSheetIndex : 0
     const margin = callbacksRef.current.getSheetMargin(sheetIdx)
     const sheetType = estimate.sheets[sheetIdx]?.type ?? ''
     const msg = `${sheetType} 마진 ${Math.round(margin)}퍼센트.`
     callbacksRef.current.addLog('assistant', msg)
     voicePlayTtsRef.current(msg)
     return true  // LLM 건너뜀
   }
   ```
