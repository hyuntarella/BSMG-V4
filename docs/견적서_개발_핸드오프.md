# 견적서 자동화 프로그램 — 개발 핸드오프

## 프로젝트 개요

엑셀 데이터를 기반으로 견적서를 자동 생성하는 프로그램. 최종 출력물은 엑셀 파일과 PDF. 견적서는 갑지 1장 + 을지 N장으로 구성되며, 을지는 공사 개수만큼 자동 생성됨 (예: 주차장/외벽/옥상 → 을지 3장).

## 갑지 (1장 고정)

전체 현장의 공사 목록과 총합계를 표시하는 표지.

### 프레임 구조
```
Frame 1 (갑지)
├── header-row (로고 왼쪽 + "견 적 서" 타이틀 가운데 + spacer 오른쪽)
├── info-row
│   ├── left-info (수신자)
│   │   ├── row-견적일: 날짜
│   │   ├── row-담당자: 이름, 연락처
│   │   ├── row-주소: 현장 주소
│   │   └── row-공사명: 공사명
│   └── right-info (공급자)
│       ├── row-상호: 상호(법인명), 사업자번호
│       ├── row-업태: 업태 및 종목, 대표자 성명
│       ├── row-대표번호: 대표번호, FAX
│       └── row-사업장주소: 사업장 주소
├── amount-block
│   ├── Frame 24: "공사금액 (부가세 별도)" 라벨 (dark bg, 흰색 텍스트)
│   └── 일금 텍스트: "일금 팔천오백칠십만원 (₩85,700,000)"
├── table
│   ├── table-header: 품명 / 규격 / 수량 / 단가 / 금액 / 비고
│   ├── table-row × N: 공사 항목 (을지 1장당 1행)
│   └── table-footer: 합계 행, (단수정리) 표기
├── special-notes
│   ├── notes-main: "특기사항" 라벨 + 본문
│   │   - 하자보수기간 5년 (하자이행증권 3년)
│   │   - 견적서 제출 30일 유효
│   └── notes-caption: * 부가가치세별도
└── brand-logos: "Brand Collaborations" + 로고 이미지들
```

## 을지 (공사 개수만큼 자동 생성)

공사 1개당 1장. 해당 공사의 상세 품목과 합계를 표시.

### 프레임 구조
```
Frame 2 (을지)
├── logo (방수명가 로고)
├── construction-name-row
│   ├── label: "공사명" (dark bg)
│   └── text: 공사명 (예: "6층 주차장 고경질 우레탄 논슬립 3mm")
├── table
│   ├── table-header (2단 병합 구조, 아래 상세 참고)
│   ├── table-row × N (품목 데이터, 가변)
│   ├── row-subtotal (소 계)
│   ├── row-overhead (공 과 잡 비, 안 전 관 리 비 — %)
│   ├── row-profit (기 업 이 윤 — %)
│   ├── row-total (계 — 항목별 소계)
│   └── row-grand-total (합 계 — bold, 비고에 "(단수정리)" 표기)
```

### 을지 테이블 헤더 — 2단 병합 구조

```
┌──────┬──────┬────┬────┬──────────┬──────────┬──────────┬──────────┬────┐
│      │      │    │    │  재료비   │  인건비   │   경비    │   합계    │    │
│ 품명  │ 규격  │단위│수량├────┬─────┼────┬─────┼────┬─────┼────┬─────┤비고│
│      │      │    │    │단가 │금액  │단가 │금액  │단가 │금액  │단가 │금액  │    │
└──────┴──────┴────┴────┴────┴─────┴────┴─────┴────┴─────┴────┴─────┴────┘
```

- 품명, 규격, 단위, 수량, 비고: 2행 높이 통으로 사용
- 재료비, 인건비, 경비, 합계: 상단 1행에 대분류명, 하단 1행에 "단가 | 금액"

### 을지 행 배분 규칙 (가변 행 방식)

빈 행 없이 데이터 개수만큼만 생성. 한 페이지 안에서 자연 배치.

- 데이터 행: 품목 개수만큼 N행
- 소 계: 1행
- 공 과 잡 비, 안 전 관 리 비: 1행 (단위 %, 수량에 요율)
- 기 업 이 윤: 1행 (단위 %, 수량에 요율)
- 계: 1행 (합계 금액 표시)
- 합 계: 1행 (bold, 비고에 "(단수정리)" 표기)

### 을지 정렬 규칙

| 컬럼 | 정렬 |
|------|------|
| 품 명 | 가운데 |
| 규 격 | 가운데 |
| 단 위 | 가운데 |
| 수 량 | 오른쪽 |
| 재료비·인건비·경비·합계 단가 | 오른쪽 |
| 재료비·인건비·경비·합계 금액 | 오른쪽 |
| 비 고 | 가운데 |

예외: 공과잡비·기업이윤 행의 "%" 수량은 퍼센트 숫자이므로 가운데 정렬.

### 페이지 분할 규칙

을지 1개가 한 페이지를 넘기는 경우는 실무상 발생하지 않음. 13~15개 이상 품목이 예상되면 PM과 협의하여 공사 단위를 분할.
